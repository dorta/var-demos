# FFT Example (GPU, CPU)

## Installation

1. Compile the C++ FFT example:

```console
$ make
```

2. Copy the following files to the module:

```console
$ scp fft run_fft.py root@<ip_address>:~
```

3. Execute the following script on the module:

```console
# python3 run_fft.py
```
